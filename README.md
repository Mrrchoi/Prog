# 깃 컨벤션
## 브렌치 생성 컨벤션
- 브랜치 이름 생성 
  - 구분/commit type/기능명_지라 이슈 넘버 
    - be/feature/login_#72
    - fe/feature/login_#99
- 형상 관리
  - main : 기준이 되는 브랜치 (재품 배포)
  - dev : 개발 브랜치, 배포전 최종 확인
  - hotfix : 버그 수정
  - feature : 기능단위 개발
- 커밋 컨벤션
  - commit type
    - Feat : 기능 추가, 수정
    - Fix : 버그 수정
    - Docs : 문서 수정
    - Refactor : 기능 고도화
    - Style : 코드 포멧팅
    - Chore : 빌드 업무 수정
    - Design : UI 관련 코드
  ````
    제목 : [BE] Feat : Add security
    본문 : 시큐리티 설정 추가
    ````
